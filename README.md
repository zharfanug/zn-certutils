# ZN Certutils

## Installation

Download the latest version and move it to your `$PATH`:

Using `wget`:

```bash
wget -O /usr/local/bin/zn-certutils https://raw.githubusercontent.com/zharfanug/zn-certutils/latest/zn-certutils.sh && chmod +x /usr/local/bin/zn-certutils
```
Using `curl`:
```bash
curl -o /usr/local/bin/zn-certutils https://raw.githubusercontent.com/zharfanug/zn-certutils/latest/zn-certutils.sh && chmod +x /usr/local/bin/zn-certutils
```
